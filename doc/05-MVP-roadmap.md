# MVP Roadmap

## Phase 1 — Concept & Documentation (1–2 weeks)
- Define attributes
- Define XP system
- Create daily log structure
- Build initial Notion/Excel prototype
- Begin testing on myself

## Phase 2 — Prototype Development (2–6 weeks)
- Create Streamlit or Flutter prototype
- Implement XP calculator
- Display character sheet
- Track daily actions
- Generate weekly summary

## Phase 3 — Pre-App Version (1–3 months)
- Introduce user accounts
- Persistent storage (SQLite / Firebase)
- Refined UI/UX inspired by RPG stat screens
- Basic animations for leveling up

## Phase 4 — Full App Development (Long-term)
- Full frontend (Flutter/React Native)
- Backend API
- Cloud deployment
- Achievement system
- Unlockable content
- Themes and customization

## Philosophy
Progress will be made through micro-tasks documented daily.  
The goal is not speed — it is consistency.
